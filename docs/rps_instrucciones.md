# Práctica 1: Juego clásico Piedra–Papel–Tijera

Archivo principal: `src/rps.py`

## Objetivos
- Implementar lógica condicional.
- Usar GitHub Copilot como apoyo (no como solución completa).

## Tareas
1. Abre el proyecto `ejercicios` en VSC.
2. Abre `rps.py` de la carpeta `src`. Esta es una bosquejo del programa a implementar. 
3. Antes de usar la IA, Revisa los comentarios, piensa lo que pide, checa los `# TODO: implementar lógica`.
4. Selecciona el bloque del TODO con el mouse, y luego da Click en Copilot Chat. Pregunta algo así:
```
Ayúdame a completar este TODO.
Quiero entender la lógica.
Explícame paso a paso que debo hacer
No quiero la solución completa, solo dame pistas.
```
5. Si se te complica, pídele el código. Analíza el resultado para ver si es lo que debe hacer.
4. Ejecutar el programa varias veces para probar que hace lo que se pide.


## Sugerencias de uso de IA
- PyLIA: para análisis, requisitos y pseudocódigo.
- TutorPy: para dudas de Python (if/else, funciones, módulos).
- GitHub Copilot: para autocompletar funciones pequeñas o docstrings.
